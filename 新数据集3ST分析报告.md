## 5. 初学者解读

### 什么是偏度和峰度？

#### 偏度（Skewness）

偏度衡量数据分布的**对称性**：

- **偏度 ≈ 0**：分布对称，类似正态分布（钟形曲线）
- **偏度 > 0**：右偏（正偏），大部分数据集中在左侧，右侧有长尾
- **偏度 < 0**：左偏（负偏），大部分数据集中在右侧，左侧有长尾

```
右偏分布示意：      左偏分布示意：
    █                   █
   ██                   ██
  ███                   ███
 █████                   ████
███████                 ██████
-------                 -------
```

#### 峰度（Kurtosis）

峰度衡量数据分布的"尖锐程度"：

- **峰度 ≈ 0**：与正态分布相似
- **峰度 > 0**：尖峰分布，数据集中在均值附近，尾部较重
- **峰度 < 0**：扁平分布，数据分散，尾部较轻

```
尖峰（峰度高）：    扁平（峰度低）：
      █               ████
     ███              █████
    █████             ██████
   ███████           ████████
```

### 偏度 > 2 意味着什么？

当偏度超过2时，表示：

1. **大部分样本集中在低值区域**：例如，icp通路的大部分样本评分接近0或负值
2. **少数极高值拉长了分布尾巴**：存在少量样本的icp评分非常高
3. **分布严重不对称**：传统的均值、标准差可能无法准确描述数据特征

**实际例子**：LMZ12939的toxic通路偏度3.07，意味着：
- 约70-80%的样本toxic评分为负值或接近0
- 只有少数样本有高toxic评分
- 这些高评分样本可能是对放化疗特别敏感或耐受的区域

### 对模型训练的影响

#### 问题1：模型倾向预测中位数

当目标值分布不对称时，模型会倾向于预测"安全"的中位数或均值，导致：
- 对低值样本预测相对准确
- 对高值样本（长尾部分）预测偏差大
- 整体损失被多数低值样本主导

#### 问题2：难以捕捉极值

极端值（如icp、toxic的高值）样本数量少，模型难以学习其特征，导致：
- 高值区域的预测精度低
- 可能漏掉重要的生物学信号

#### 解决方案：使用鲁棒损失函数

**推荐使用Huber Loss**：

```python
import torch.nn as nn

# Huber Loss对异常值更鲁棒
loss_fn = nn.HuberLoss(delta=1.0)
```

Huber Loss的特点：
- 对小误差使用平方损失（类似MSE），保证收敛稳定
- 对大误差使用线性损失，减少异常值的影响
- 适合处理偏态分布和存在离群值的数据

### 三个患者的"个性差异"（病理异质性）

不同患者的肿瘤微环境不同，同一通路在不同患者中的表达模式也不同：

| 患者 | 主要特征 | 临床可能解读 |
|-----|---------|-------------|
| HYZ15040 | 分布均匀，接近正态 | 肿瘤微环境相对均一，可能为早期或分化较好的肿瘤 |
| JFX0729 | 两极分化（mhc左偏，icp/toxic右偏） | 免疫浸润不均，可能存在免疫荒漠区和免疫炎症区 |
| LMZ12939 | icp/toxic极端偏态 | 免疫检查点和毒性特征高度异质，可能对免疫治疗反应复杂 |

**启示**：联合训练时需注意患者间异质性，可考虑：
- 使用患者ID作为条件输入
- 分患者评估模型性能
- 对极端分布的通路使用专门的损失函数

### CSV样本数与Patch数量不一致的原因

| 数据项 | 数量 | 说明 |
|-------|------|------|
| CSV总样本数 | 71,903 | 空间转录组所有检测点 |
| Patch总文件数 | 18,006 | 有效图像Patch |

**不一致原因**：

1. **组织覆盖问题**：部分空间转录组点位落在组织边缘或空白区域，无有效图像
2. **图像质量筛选**：低质量图像（模糊、染色异常）被过滤
3. **split采样策略**：split.py可能按一定比例采样，而非使用全部样本

## 7. 注意事项

### 训练数据匹配

- CSV样本数远多于Patch文件数
- **训练时必须按Patch文件名匹配CSV标签**
- 建议在数据加载时建立Patch文件名到CSV行的映射

### 极端分布处理

**LMZ12939的icp和toxic通路分布极端**（峰度>10），建议：

```python
# 方案1：使用Huber Loss
loss_fn = nn.HuberLoss(delta=1.0)

# 方案2：对极端通路使用加权损失
weights = {'icp': 2.0, 'toxic': 2.0}  # 增加权重

# 方案3：对目标值进行变换（如对数变换）
target_transformed = torch.log1p(target - target.min() + 1)
```

### 患者异质性

三个患者的通路分布存在显著差异，联合训练时需注意：

1. **分患者评估**：训练后对每个患者单独评估性能
2. **分层抽样**：确保训练/验证集中各患者比例一致
3. **患者ID编码**：可将患者ID作为条件输入，让模型学习患者间差异

### 数据使用建议

| 场景 | 建议 |
|-----|------|
| 单患者模型 | 使用HYZ15040（分布最均匀） |
| 多患者模型 | 使用全部数据，注意分层抽样 |
| 特定通路研究 | 对icp/toxic使用鲁棒损失函数 |
| 迁移学习 | 先在均匀分布通路预训练，再finetune极端通路 |
