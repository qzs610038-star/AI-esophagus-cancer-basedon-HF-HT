## 二、逐实验分析

### Exp-A（仅H&E增强）— ❌ 无效

- Val PCC = 0.3948，甚至略低于基线（Δ=-0.0015，-0.4%）
- 过拟合差仅 0.1782（但这是因为Train PCC本身被压低，并非泛化好）
- **根因**：HED色彩空间全局扰动 + 静态预提取 → UNI的DINOv2预训练已内置强颜色不变性，此类增强几乎无信息增益

### Exp-B（仅MixUp）— ✅ 小幅有效

- Val PCC = 0.4182，提升 +0.0218（+5.5% vs 基线）
- 过拟合差 0.2096，略低于基线的 0.2569
- **根因**：MixUp在token嵌入层引入的隐式正则化起到了label smoothing效果，减轻过拟合

### Exp-C（联合）— ⚠️ 被拖累

- Val PCC = 0.4108，高于基线但低于纯MixUp
- H&E增强的噪声样本稀释了MixUp的正则化效果
- **结论**：两种增强联合使用在当前配置下无协同效应

## 六、操作合理性评估

### ✅ 做得好的
- 消融实验设计（A/B/C三组对照）逻辑正确
- 独立保存训练历史、per-pathway PCC、可视化 → 可追溯
- 数据集适配器完全向后兼容
- 多维度评估（PCC + R² + MAE + per-pathway）

### ⚠️ 关键问题

**1. 超参数混杂（最严重）**
- 基线 Dropout=0.5，AugMix实验 Dropout=0.3 → 正则化强度不同
- 基线 LR=5e-05，AugMix LR=1e-04 → 收敛路径不同
- 无法判断性能差异来自增强方法还是超参数

**2. H&E增强方式与UNI Backbone性质重叠**
- UNI在TCGA/CPTAC百万级H&E上预训练，内置强颜色不变性
- HED空间全局偏移 → 模拟不同染色条件 → UNI对此天然鲁棒
- 静态预提取（固定3变体）→ 不是每次epoch随机生成 → 多样性不足

**3. Token层MixUp语义存疑**
- patch token经过自注意力编码后是上下文表示，混合后不映射到真实组织形态
- 效果可能来自隐式正则化（类似label smoothing），而非"数据增强"

## 七、优化建议（按优先级）

### 🥇 Tier 1：消除混杂，重建基线（明天可做）

```yaml
# 推荐统一超参数
learning_rate: 5e-05
dropout: 0.5
scheduler_patience: 8
epochs: 150
```

- 增跑无增强对照组（相同超参数，关闭AugMix和MixUp）
- MixUp alpha 扫参：α ∈ {0.1, 0.2, 0.4, 0.6, 0.8}（当前α=1.0 = Beta(1,1) 过于激进）

### 🥈 Tier 2：改进增强策略

**a) Stain Normalization（思路反转）**

不增加染色多样性，而是消除域差异。Macenko/Vahadane预处理使JFX/LMZ/HYZ三患者的H&E染色风格统一，预测跨患者PCC提升幅度约 +0.03~0.08：

```
JFX切片 → Macenko归一化 → 统一染色空间 → UNI提取token
LMZ切片 → Macenko归一化 → 统一染色空间 → UNI提取token
HYZ切片 → Macenko归一化 → 统一染色空间 → UNI提取token
```

**b) 在线增强（替代静态预提取）**

改为每个batch随机在线增强：每次从HED/HLS/HSV中随机选一个空间，随机生成偏移参数 → 无限变体 vs 固定3变体

**c) CutMix替代/配合MixUp**

在token序列维度做CutMix（随机截取一段token替换为另一个样本的对应段），保留局部空间连续性

### 🥉 Tier 3：架构优化（中期）

| 方向 | 当前 | 建议 | 预期 |
|---|---|---|---|
| 编码器深度 | 1层 | 2~4层 | +0.03~0.06 PCC |
| 模型维度 | d_model=1024 | 2048 | +0.02~0.04 PCC |
| 位置编码 | 固定sin/cos | 可学习+相对位置 | 空间建模提升 |
| 多尺度token | 65个(16×16) | 加粗粒度token | 捕获大范围组织模式 |
| UNI微调 | 冻结 | LoRA/Adapter微调后几层 | **可能是最大提升** |

## 八、总结一览

| 维度 | 结论 |
|---|---|
| **实验设计** | 消融思路正确 ✅，超参数混杂 ⚠️ |
| **H&E增强** | 方向对（HED空间），方式错（静态+UNI不变性重叠）→ 无效 ❌ |
| **MixUp** | 有效但幅度有限 +5.5% ✅，α=1.0 需调参 ⚠️ |
| **根本瓶颈** | 非数据增强不足，而是token序列方案（7.7M）表达力天花板。距完整HisToGene（0.577）差 30%+ |
| **最强单步提升** | Stain Normalization 预处理（消除域偏移，而非增加多样性） |
| **明天优先** | 统一超参数重跑基线 + MixUp α扫参 |
