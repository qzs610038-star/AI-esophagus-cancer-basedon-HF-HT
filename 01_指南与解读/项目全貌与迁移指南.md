## 1. 项目背景与目标

- **项目名称**：PFMval_new (Pathology Foundation Model Validation - new)
- **研究领域**：食管癌（Esophageal Cancer，**非肺癌**）H&E 病理切片 + 空间转录组联合分析
- **核心任务**：基于 H&E 图像 patch 预测对应空间位置的 **30 条基因通路 ssGSEA 活性评分**
- **标签处理**：ssGSEA 原始得分经过 **Z-score 标准化**
- **评估指标**：PCC（Pearson 相关系数），逐通路计算后取均值
- **工作目录**：`d:\AI空间转录病理研究\PFMval_new`

## 10. 已知坑点与解决方案

### 10.1 OmiCLIP 加载

**问题**：`from loki.predex import OmiCLIP_Predictor` 报错 `ImportError`。
**原因**：loki==0.0.1 包中 `loki.predex.OmiCLIP_Predictor` **不存在**。
**解决**：跳过 loki 高层 API，直接用 `open_clip` 加载：
```python
import open_clip, torch
model = open_clip.create_model('coca_ViT-L-14')
ckpt = torch.load('pretrained_omiclip/checkpoint.pt', map_location='cpu')
model.load_state_dict(ckpt['state_dict'])  # 注意 key 是 state_dict（非 model）
# 输出
tokens = model.visual(x)[1]               # [B, 255, 768] — token 序列
cls    = model.encode_image(x)            # [B, 768]
```

### 10.2 PyTorch 2.0.1 中文路径写文件失败

**问题**：在 loki_env（PyTorch 2.0.1）中 `torch.save(tensor, "中文路径/file.pt")` 报错。
**原因**：PyTorch 2.0.1 在 Windows 下对中文路径的 fopen 处理有 bug。
**解决**：先序列化到 `io.BytesIO`，再用 Python 原生 `open()` 写入：
```python
import io, torch
buf = io.BytesIO()
torch.save(tensor, buf)
with open(chinese_path, 'wb') as f:
    f.write(buf.getvalue())
```

### 10.3 HuggingFace 国内下载卡住

**问题**：`huggingface_hub.snapshot_download(...)` 在国内长时间无响应。
**原因**：huggingface.co 在国内不可达或极慢，Python 库底层重试导致卡死。
**解决**：用 `curl.exe` 配合 `hf-mirror.com`：
```powershell
curl.exe -L -o pretrained_omiclip\checkpoint.pt `
  https://hf-mirror.com/WangLab/OmiCLIP/resolve/main/checkpoint.pt
```

### 10.4 dataset_uni_tokens.py 1536 维硬编码

**问题**：直接复用 `UniTokensDataset` 加载 OmiCLIP 特征 `[255,768]` 时报断言错误。
**原因**：`dataset_uni_tokens.py` 中存在 `assert feat.shape[-1] == 1536` 断言。
**解决**：为 OmiCLIP 创建独立的 `OmiCLIPTokensDataset`，断言改为 `768`。

### 10.5 conda activate 在 PowerShell 报错

**问题**：`conda activate loki_env` 报 `CommandNotFoundError`。
**原因**：PowerShell 默认未注入 conda hook。
**解决**：先执行 hook：
```powershell
(& "D:\miniconda\Scripts\conda.exe" "shell.powershell" "hook") | Out-String | Invoke-Expression
conda activate loki_env
```

### 10.6 GPU 属性名不一致

**问题**：`torch.cuda.get_device_properties(0).total_mem` 报 `AttributeError`。
**原因**：属性名为 `total_memory`（不是 `total_mem`）。
**解决**：使用 `total_memory`。

### 10.7 受保护目录 dataset_name 错误

**问题**：`histogene/` 内脚本默认 dataset_name 为旧值，导致路径错误。
**解决**：通过命令行参数 `--dataset_name <正确值>` 覆盖，**不修改源码**。

### 10.8 可视化结果列名不匹配

**问题**：`visualize_results.py` 报找不到通路列。
**原因**：`predictions.csv` 列名未严格使用 `true_xxx` / `pred_xxx`。
**解决**：训练脚本写入 CSV 时严格按此命名。
