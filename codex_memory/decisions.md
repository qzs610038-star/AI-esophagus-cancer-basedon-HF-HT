## 2026-05-30：UNI2-h 在线 LoRA/渐进解冻冒烟测试通过，启动 50 epoch 训练

决策：在线 LoRA 训练方向代码落地后立即冒烟测试，通过后启动单患者 50 epoch 快速验证。

关键事实：
- 冒烟测试 1 epoch：Val PCC=0.5379（vs frozen 基线 0.5227），显存 4.5GB/8.2GB 安全
- 跨阶段 LoRA merge 机制已验证（merge 前后前向输出差异=0）
- 50 epoch 训练 5.30 15:10 启动，预计 3.5h 完成
- Python `-u` 参数必须加——后台训练无此标志则 stdout 完全缓冲，无法监控
- HF client 网络不可达时自动重试 5 次（~30s），仍从本地缓存加载

判断：
- LoRA 方向值得推进——1 epoch 即超 frozen 基线说明 backbone 存在任务适配空间
- 但单患者验证只是第一步，跨患者 Fold1 才是真正门槛
- 决策门槛：Fold1 PCC 提升 ≥ 0.01 才进入三折 CV
