### 2.2 模型深度分析

#### ⭐ Hist2ST（2023，Nature Machine Intelligence）
- **方法精髓**：CNN提取HE tile特征 → Transformer学习空间邻域依赖 → 预测每个spot的基因表达
- **优势**：(1)首次将空间坐标编码进Transformer实现邻域感知 (2)代码完整开源 (3)在乳腺癌HER2+数据上验证充分
- **局限**：依赖Visium级spot（100μm），无法到单细胞精度；Transformer在WSI级别扩展有瓶颈
- **对课题的启示**：**管线起点模板**——可直接复用其空间Transformer思路，但在食管癌组织上需验证泛化性

#### ⭐ EGN（Exemplar Guided Network, 2023 WACV / EGN-v2 2026）
- **方法精髓**：双分支——(1)局部图像编码器 (2)全局exemplar引导分支；通过可学习的exemplar特征匹配来约束表达预测
- **优势**：exemplar机制天然具备**可解释性**（每个预测可由哪个exemplar驱动追溯）；WACV严格双盲评审
- **局限**：exemplar选择敏感，需充分覆盖组织多样性
- **对课题的启示**：**最适合食管癌场景**——exemplar可以提供"该区域最像哪种已知分子亚型的食管癌组织"的解释，成果可视化强

#### ⭐ PEaRL（Pathway-Enhanced Representation Learning, 2026 WACV）
- **方法精髓**：在基因表达预测之外，额外预测**通路活性**作为辅助监督信号，迫使模型学习生物学可解释的表示
- **优势**：通路层面比单基因更稳定、更生物学可解释；WACV最新接收（2026）
- **局限**：依赖预测的通路数据库（KEGG/MSigDB）；实际增益vs额外复杂度需权衡
- **对课题的启示**：**与食管癌预后高度契合**——食管癌关键通路（p53、Wnt、Notch、EMT）可作为通路监督目标，预测的"通路热图"就是生物学解释

#### ⭐ Path2Space（2025 bioRxiv，未正式发表）
- **方法精髓**：DINOv2基础模型提取HE特征 → 多实例学习聚合 → 预测基因表达+患者预后
- **优势**：(1)DINOv2带来的强大视觉特征 (2)端到端预后预测 (3)可处理WSI大片
- **局限**：bioRxiv预印本未peer-review；多实例学习弱化了空间精细度
- **对课题的启示**：**预后预测端参考**——如果目标是临床预后预测，可以直接对标

#### ⭐ FmH2ST（Foundation Model-based, 2024）
- **方法精髓**：UNI作为冻结backbone提取病理特征 → 轻量adapter微调 → 预测基因表达
- **优势**：借力大规模预训练（UNI基于10万+WSI训练），泛化性好；adapter轻量高效
- **局限**：UNI模型较封闭（未完全开源权重）
- **对课题的启示**：**技术路线可选方案**——如果选择UNI/Virchow作为backbone，可参考其adapter设计

#### ⭐ GHIST（2024，单细胞分辨率）
- **方法精髓**：ViT+图注意力网络，在单细胞级别预测空间基因表达
- **优势**：目前分辨率最高的方法之一；利用图结构建模细胞间通讯
- **局限**：依赖GNN，在大型WSI上计算开销大
- **对课题的启示**：**最高空间精度方案**——如果食管癌的TME亚群（如CD8+T细胞空间分布）是关键变量

## 三、病理基础模型对比（02_病理基础模型_FoundationModels/）

| 模型 | 机构 | 训练数据 | 参数量 | 开放程度 |
|------|------|---------|:----:|:----:|
| **UNI** (Chen et al., Nature Medicine 2024) | Harvard | 10万+ WSI | ViT-L(307M) | ✅ 权重开源 |
| **CTransPath** (Wang et al., MedIA 2022) | 清华 | 3.2万 WSI, 1500万patch | ViT-S(22M) | ✅ 完全开源 |
| **Prov-GigaPath** (Xu et al., Nature 2024) | Providence | 17万 WSI | LongNet架构 | ✅ 权重/代码开源 |
| **Virchow** (Vorontsov et al., Nature Medicine 2024) | Paige AI | 150万 WSI | ViT-H(632M) | ❌ 商业限制 |
| **CONCH** (Lu et al., Nature Medicine 2024) | Harvard | 42万病理-文本对 | 视觉语言模型 | ⚠️ 受限使用 |

### 推荐选择策略（对食管癌ST映射课题）

```
优先级：Prov-GigaPath > UNI > CTransPath > CONCH > Virchow
```

- **Prov-GigaPath**：2024 Nature，17万WSI训练，支持整张WSI的LongNet架构，完全开源，是当前最佳
- **UNI**：Nature Medicine背书，权重开源，与FmH2ST经验可复用的直接参考
- **CTransPath**：轻量(22M参数)，完全开源，可作为快速实验baseline
- **见`02_病理基础模型_FoundationModels/`文件夹内5篇原文详细阅读**

## 四、MIL框架选型（03_多实例学习MIL与WSI分类/）

| 方法 | 年份 | 核心创新 | 适用场景 |
|------|------|---------|---------|
| **ABMIL** (Ilse et al., 2018) | 2018 | 注意力池化，首次引入MIL可解释性 | 基础baseline |
| **CLAM** (Lu et al., 2021) | 2021 | 聚类约束注意力，弱监督病理SOTA | 多分类/亚型 |
| **TransMIL** (Shao et al., 2021) | 2021 | Transformer+PPEG位置编码，WSI长序列 | 大规模WSI |
| **DSMIL** (Li et al., 2021) | 2021 | 双流MIL，实例级和bag级联合 | 高精度WSI |
| **Pan-cancer outcome** | 2024 | 统一框架预测多癌种预后 | 泛癌预后 |

### 推荐
- **Base选择**：CLAM（稳定、可解释、代码成熟）
- **精度优先**：TransMIL（Transformer全局建模）+ DSMIL（双流互补）
- **本课题特殊**：HE→ST映射是**回归任务**而非分类，MIL框架需改造：将分类头换成回归头（预测每个spot的基因表达向量）

## 五、模型选型综合建议（核心推荐方案）

### 方案A：稳健推进型（⭐⭐⭐⭐⭐ 推荐优先采用）

```
Backbone: Prov-GigaPath (或UNI) 
↕ 冻结/微调策略参考 FmH2ST
Mapping头: EGN exemplar引导 + PEaRL通路监督
↕ 利用exemplar可解释性 + 通路生物学约束
空间增强: Hist2ST Transformer邻域编码
预后端: Path2Space端到端预后预测
```

**理由**：
1. **Prov-GigaPath**是2024 Nature最新最强开源病理基础模型
2. **EGN**的exemplar机制让食管癌分子亚型可视化——医生可直接理解
3. **PEaRL**的通路预测天然适合食管癌（p53突变率>80%，通路异常是核心机制）
4. **Hist2ST**的空间Transformer是唯一经过充分验证的邻域建模方案
5. 四者代码均不同程度开源，具备工程可行性

### 方案B：前沿创新型（⭐⭐⭐⭐）

```
Backbone: DINOv2（Path2Space验证过的视觉特征）
Mapping: GHIST的单细胞级图注意力
通路: PEaRL通路监督
预后: 自定义Cox回归头
```

**优势**：空间分辨率最高（单细胞级），解释粒度最细  
**风险**：GHIST计算开销大，食管癌数据适配性未知

### 方案C：轻量快速型（⭐⭐⭐，适合预实验/baseline）

```
Backbone: CTransPath (22M参数，快速迭代)
Mapping: Hist2ST直接移植
预后: CLAM式注意力聚合
```

**优势**：快速出结果，做消融实验的baseline
