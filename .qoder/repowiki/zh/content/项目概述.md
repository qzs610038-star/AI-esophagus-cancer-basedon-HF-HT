# 项目概述

## 项目目标

基于食管癌 H&E 病理切片（patch 级，224x224），使用深度学习模型预测 **30 条基因通路的 ssGSEA 活性评分**（Z-score 标准化）。核心评估指标为 **PCC**（Pearson 相关系数，逐通路计算后取均值），辅助指标包括 MAE、R²。

## 数据集概况

3 个食管癌患者的空间转录组 + H&E 病理数据：

| 患者 ID | Patch 数量 | 用途 |
|---------|-----------|------|
| HYZ15040 | 2,655 | 单患者实验 / 跨患者测试集 |
| JFX0729 | 7,838 | 跨患者训练 |
| LMZ12939 | 7,513 | 跨患者训练 |

数据路径：`data_new_3ST/patch_noov_spilt/{patient}_noov_split/`（typo "spilt" 是原始拼写，不可修正）。标签文件：`data_new_3ST/ssGSEA_zscore/{patient}_ssGSEA_zscore.csv`。

## 当前模型矩阵

| 模型 | 状态 | 特征维度 | 最佳 PCC |
|------|------|---------|----------|
| **HisToGene-UNI Token** | **主力** | UNI2-h tokens [265, 1536] | 0.5217 (单患者) / 0.3812 (跨患者三折平均) |
| **HisToGene-UNI Token + AugMix** | 活跃 | UNI2-h tokens [265, 1536] + 数据增强 | 0.4142 (跨患者 Fold1) |
| EGN-v2+UNI | 活跃 | UNI2-h CLS (1536d) | 0.195 (跨患者 PCC) |
| HisToGene 原版 (ViT) | 活跃(基线) | ViT 224x224 图像输入 | — |
| HisToGene-UNI+GAT | 实验完成 | UNI2-h + 图注意力 | 提升不显著 (-0.27%) |
| OmiCLIP | 新增 | coca_ViT-L-14 [255, 768] | 训练验证中 |
| Virchow2 | 新增(已部署) | — | 待训练 |

淘汰模型：EGN-v1（所有对比分析中排除）。

## 核心铁律

1. **受保护目录禁改**：`histogene/`、`egnv1/`、`egnv2/` 下文件禁止修改，适配通过根目录独立文件实现
2. **路径 typo 不修正**：`patch_noov_spilt` 是原始拼写，所有引用保持一致
3. **val_loss 选最优 epoch**：非 val_pcc 最大
4. **predictions.csv 列名**：`true_xxx` / `pred_xxx` 格式（visualize 依赖）
5. **路径统一管理**：所有脚本通过 `config_utils` 函数获取路径，禁止硬编码绝对路径
6. **三折交叉验证**：Fold1(JFX+LMZ→HYZ)、Fold2(HYZ+LMZ→JFX)、Fold3(HYZ+JFX→LMZ)
7. **训练输出不覆盖历史**：每次训练保存至 `{model_dir}/checkpoints/results_vis/{dataset}_{timestamp}/`
