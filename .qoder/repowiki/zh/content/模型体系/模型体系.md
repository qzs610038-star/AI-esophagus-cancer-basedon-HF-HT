## 活跃模型架构要点

### HisToGene 原版（ViT 端到端）

- 输入：224x224 PNG patch + 坐标位置编码（X/Y 独立 Embedding）
- 架构：Patch Embedding + CLS Token + ViT 位置嵌入 + 多层 Transformer Block + 回归头
- Transformer：depth=8, heads=16, dim=1024, mlp_dim=2048
- 回归头：LayerNorm + Linear + GELU + Dropout + Linear
- 空间位置编码：分离 X/Y 嵌入加到 CLS token 上，坐标归一化到 [0, n_pos-1]
- 损失函数：HuberLoss (delta=1.0)

### HisToGene-UNI Token（主力）

- 输入：UNI2-h 提取的 token 序列 [265, 1536]（含 CLS + 256 patch tokens）
- 特征提取器：UNI2-h 冻结参数，预提取并缓存为 .pt 文件
- 回归头：HisToGene Transformer 架构（无需图像编码部分），仅处理 token 序列
- 优势：参数量仅 ~4.0M，训练速度快，跨患者泛化能力最强
- 训练脚本：`train_histogene_uni_tokens.py`（基础版）、`train_histogene_uni_tokens_augmix.py`（AugMix 增强版）

### HisToGene-UNI Token AugMix（主力）

- 与 UNI Token 基础版相同架构，但训练数据使用 AugMix 增强特征
- AugMix：对原始图像应用多种数据增强组合后提取 UNI token，生成多样化训练样本
- 增强特征由 `extract_uni_tokens_augmented.py` 预提取

### EGN-v2+UNI

- 输入：UNI2-h CLS token [1, 1536]（仅全局特征）
- 架构：特征投影层 + 图卷积层（GraphSAGE）+ Exemplar 融合 + MLP 回归头
- 图构建：基于空间坐标距离的半径邻居搜索
- 优势：参数量小（~2.8M），显存友好，图结构捕获空间依赖

### HisToGene-UNI+GAT（实验完成）

- 架构：UNI token 序列 + GAT（图注意力网络）替代标准 Transformer
- 图结构：KNN 特征相似性图
- 结论：GAT 引入图结构后 PCC 下降 0.27%，提升不显著，暂不推荐

### OmiCLIP (Loki)（新增）

- 骨干：coca_ViT-L-14（OpenAI CLIP 风格多模态模型）
- 输出：256 patch tokens + 1 CLS token，每 token 768 维
- 特征提取脚本：`extract_omiclip_features.py`
- 训练脚本：`train_histogene_omiclip.py`

### Virchow2（新增）

- 骨干：Virchow2（病理专用大模型）
- 特征提取脚本：`extract_virchow2_tokens.py`
- 训练脚本：`train_histogene_virchow2_tokens.py`

| Backbone | 来源 | 输出维度 | Token 数 | 特点 | 适用场景 |

| ResNet50 | torchvision | 2048 | 1（全局池化） | 经典 CNN，EGN-v1/v2 使用 | EGN-v1 已淘汰 |

## 当前最佳性能

| 场景 | 最佳模型 | PCC | 备注 |
|------|----------|-----|------|
| 单患者（HYZ15040） | HisToGene-UNI Token | 0.5217 | 训练/验证同患者 |
| 跨患者三折平均 | HisToGene-UNI Token | 0.3812 | 2患者训 -> 1患者测 |
| 跨患者 Fold1 | HisToGene-UNI Token AugMix | 0.4142 | JFX+LMZ -> HYZ |
| 跨患者（EGN体系） | EGN-v2+UNI | 0.195 | 跨患者泛化能力弱于 HisToGene |

跨患者训练比单患者衰减约 -31.7% 至 -77.2%，这是当前核心挑战。UNI 特征对 EGN-v2 提升显著（+28% ~ +50%），对 HisToGene 提升有限（+2.3% ~ +11.8%）。

## 损失函数与优化器（通用）

| 组件 | 配置 |
|------|------|
| 损失函数 | MSELoss / HuberLoss(delta=1.0) |
| 优化器 | AdamW (weight_decay=1e-4) |
| 学习率调度 | ReduceLROnPlateau (mode=min, factor=0.5, patience=5) |
| 早停 | patience=10，监控 val_loss（非 val_pcc） |
| 混合精度 | AMP GradScaler（仅 CUDA） |
| 梯度裁剪 | max_norm=1.0 |
